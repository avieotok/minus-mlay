# מנקה גיליונות

אפליקציית ווב להעלאת קובץ אקסל, ניקוי תוכן, והורדת קובץ נקי — הכול רץ בדפדפן, בלי שרת. הקובץ לא עוזב את המחשב שלך. ספריית האקסל מוטמעת בקובץ — האפליקציה עובדת גם ללא אינטרנט.

## יכולות
- בחירת גיליון (לשונית) לעבודה
- מחיקת מילים/ביטויים מעמודות נבחרות, עם בחירת מיקום בתא (תחילה / אמצע / סוף / כל מקום)
- חיפוש והחלפה (כמה זוגות בבת אחת)
- הוספת טקסט בתחילת או בסוף התא
- מחיקת עמודות שלמות
- מחיקת שורות לפי ערך, שורות ריקות, סימון ידני, או שורות כפולות
- איחוד רשומות כפולות לפי עמודת מפתח (סכום כמויות, איחוד מיקונים, ספירה ועוד)
- תצוגה חיה של כל שינוי + ייצוא ל‑XLSX או CSV

## קבצים
| קובץ | תפקיד |
|------|-------|
| `index.html` | האפליקציה עצמה (קובץ עצמאי) |
| `favicon.svg`, `icon-180.png`, `icon-192.png`, `icon-512.png` | אייקונים |
| `manifest.webmanifest` | מאפשר התקנה למסך הבית עם אייקון |

> האייקון מוטמע גם בתוך `index.html`, אז הוא יופיע בלשונית הדפדפן גם אם פותחים את הקובץ לבדו.

## העלאה ל‑GitHub

### דרך הממשק של GitHub (הכי פשוט, בלי התקנות)
1. היכנס ל‑https://github.com והתחבר.
2. לחץ על **+** בפינה הימנית העליונה → **New repository**.
3. תן שם (למשל `excel-cleaner`), בחר **Public**, ולחץ **Create repository**.
4. במסך הריק לחץ **uploading an existing file**.
5. גרור לשם את **כל** הקבצים מהחבילה (`index.html` והאייקונים והמניפסט).
6. למטה לחץ **Commit changes**.

### דרך שורת הפקודה (Git)
```bash
git init
git add .
git commit -m "מנקה גיליונות"
git branch -M main
git remote add origin https://github.com/USERNAME/excel-cleaner.git
git push -u origin main
```
החלף `USERNAME` בשם המשתמש שלך.

## כתובת חיה עם GitHub Pages
כדי שהאפליקציה תהיה זמינה ככתובת אינטרנט שאפשר לפתוח מכל מכשיר:
1. בריפו: **Settings** → בתפריט הצד **Pages**.
2. תחת **Source** בחר **Deploy from a branch**.
3. בחר ענף **main** ותיקייה **/ (root)**, ולחץ **Save**.
4. אחרי דקה‑שתיים תופיע כתובת בסגנון:
   `https://USERNAME.github.io/excel-cleaner/`

מכיוון שהקובץ נקרא `index.html`, הוא ייפתח אוטומטית בכתובת הזו.

## הוספה למסך הבית (אייקון אפליקציה)
פותחים את הכתובת החיה בדפדפן הנייד → תפריט → **הוסף למסך הבית**. בזכות `manifest.webmanifest` האפליקציה תיפתח במסך מלא עם האייקון.
